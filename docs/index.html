Generator książek low-content na Amazon KDP

Projekt w Pythonie do generowania: krzyżówek (10x10), sudoku (łatwe/średnie), kolorowanek (geometria/mandale) oraz notatników/journali z personalizowaną okładką. Pliki wyjściowe zapisywane są jako PDF w popularnych wymiarach KDP (np. 6x9, 8.5x11).

Jak uruchomić lokalnie (CLI)

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m kdp_generator.cli sudoku --pages 5 --difficulty easy --trim 8.5x11 --out samples/sudoku.pdf
python -m kdp_generator.cli crossword --lang pl --trim 8.5x11 --out samples/crossword.pdf
python -m kdp_generator.cli coloring --kind mandala --pages 20 --trim 8.5x11 --out samples/coloring.pdf
python -m kdp_generator.cli notebook --title "Mój dziennik" --style lined --pages 120 --trim 6x9 --out samples/notebook.pdf

Interfejs web (lokalnie)

Prosty serwer Flask z formularzami generującymi PDF:

FLASK_APP=app.py flask run

Następnie otwórz: http://localhost:5000.

Licencja i uwagi

  • Wygenerowane pliki PDF są gotowe do druku. Upewnij się, że wybrałeś odpowiedni rozmiar (np. 6x9 dla notatników, 8.5x11 dla kolorowanek).
  • Listy słów zawierają ok. 100 haseł (PL/EN). Krzyżówka to wypełnianka 10x10.
